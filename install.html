Cómo instalar AlyaBot

Sigue estos pasos para instalar AlyaBot en tu sistema:

1. Clonar el repositorio

git clone https://github.com/Elpapiema/AlyaBot_MD.git

2. Instalar dependencias

cd AlyaBot_MD && npm install

3. Configurar el bot

Debes configurar las credenciales en el archivo .env y escanear el código QR.

4. Iniciar el bot

npm start
