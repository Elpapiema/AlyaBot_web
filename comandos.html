Lista de Comandos de AlyaBot

Aquí tienes la lista de comandos organizados por categoría.

🎮 Generales

  • .menu - Muestra el menú de comandos
  • .ping - Verifica el estado del bot
  • .info - Muestra información sobre el bot

🎭 Diversión

  • .chiste - Cuenta un chiste
  • .meme - Envía un meme aleatorio
  • .8ball - Responde preguntas al azar

🔧 Herramientas

  • .sticker - Convierte imágenes en stickers
  • .traducir - Traduce texto a otro idioma
  • .qr - Genera códigos QR

🔐 Administración

  • .ban - Expulsa a un usuario del grupo
  • .promote - Asciende a un usuario a admin
  • .demote - Degrada a un admin

📌 Para más detalles, revisa la documentación completa.
